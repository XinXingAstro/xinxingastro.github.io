Spring Web MVC是一个原生的构建在Servlet API上的Web框架，很早就被包含在Spring框架之中。Spring Web MVC最主要的两个作用是：1. 处理从客服端发来的请求，从请求中接收传入的参数。2. 将处理后的结果数据返回给页面。
